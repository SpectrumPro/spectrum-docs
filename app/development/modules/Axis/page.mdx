# Axis Input System

<Callout type="warning">

</Callout>

*Introduction & Overview*

The **Axis Input System** is a modular, protocol-agnostic input processing engine designed for real-time control.  

It is capable of ingesting input streams from **HID devices, MIDI controllers, serial interfaces, and network sources**, converting them into a unified event model and relaying that information on to controled parameters.
