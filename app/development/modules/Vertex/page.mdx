# Vertex

<Callout type="warning">
    Vertex, and its accompanying documentation are still in beta
</Callout>

*Introduction & Overview*

The **Vertex** module is the UI system powering all Spectrum GUI applications, 

It handles **window management, UI layout persistence** saving and restoring layouts to and from disk. And ships with a comprehensive library of **UI components, icons, and styles**.
