# Vertex

<Callout type="warning">
    Vertex, and its accompanying documentation are still in beta
</Callout>

*Introduction & Overview*

The **Vertex** module is the UI system powering all Spectrum GUI applications, providing a complete front-end layer built specifically for the Spectrum Lighting Engine.  

It handles **window management, UI layout persistence** — saving and restoring layouts to and from disk — and ships with a comprehensive library of **UI components, icons, and styles** needed to build consistent, cohesive interfaces across the Spectrum ecosystem.
