# Utils

<Callout type="warning">
    Utils, and its accompanying documentation are still in beta
</Callout>

*Introduction & Overview*

The **Utils** module is a shared foundation, providing common functionality consumed across every component and subsystem.  

It bundles **utility scripts, custom types, and abstract base classes** covering the recurring patterns and primitives that would otherwise be duplicated throughout the codebase.
