*Introduction & Overview*

The **Axis Input System** is a modular, protocol-agnostic input processing engine designed for real-time control surface integration.  

It is capable of ingesting input streams from **HID devices, MIDI controllers, serial interfaces, and network sources**, normalising them into a unified event model and relaying that information on to connected control targets.
